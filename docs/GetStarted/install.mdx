### Vous avez choisi d'utiliser le code source de l'application

  <p>Un choix honorable mais risquer.</p>
  Avant de continuer veillez à remplir ces prérequis :
  <ul>
    <li> Avoir <a href="https://git-scm.com/">git</a> sur son ordinateur </li>
    <li> Avoir une installation de <a href="https://nodejs.dev/">node</a> v14 ou plus</li>
    <li> Avoir <a href="https://yarnpkg.com/">Yarn</a> ou <a href="https://www.npmjs.com/get-npm">Npm</a> (de préférence Yarn)</li>
  </ul>
  <p>Prêt à commencer ?</p>

#### 1. Récupération du code source

  ```shell
    git clone https://github.com/batleforc/UnlabeledProject.git
    cd UnlabeledProject
  ```

:::info

Première étape, vous venez de récupérer le code source de cette solution sans titre et de vous déplacer dans le dossier les contenant

:::

### Que c'est t'il passer ?

Pourquoi avoir choisis un client sous cette forme ?

Deux chose sont a savoir:

- Le client est uttilisable via votre navigateur ou le client, certaine fonctionnalité telle que la persistance du client en ten que service d'arriére plan (que je sais possible) est une fonctionalité a venir. l'uttilisateur n'aura a terme qu'a lancer son navigateur ou a juste double cliquer sur l'icone dans le volet de notification/ ou encore a relancer l'executable
- A terme une version serveur sera disponible.
